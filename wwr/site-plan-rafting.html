Rafting Site Plan

Overview

Purpose

The purpose of this website is to provide information about rafting adventures, including trip details, safety guidelines, booking information, and more.

Audience

The target audience for this website includes adventure seekers, outdoor enthusiasts, and anyone interested in experiencing thrilling rafting trips.

Style Guide

Color Palette

The color palette will consist of blues and greens to reflect the natural river environment.

  • primary-color: #396E94 (Dark Blue)
  • secondary-color: #E7C24F (Red)
  • accent1-color: #A43312 (White)
  • accent2-color: #40916C (Green)

Typography

Headings Example Font

We will use 'Roboto' for headings, which is a sans-serif font.

Normal Text/Paragraph Example Font

For normal text and paragraphs, we will use 'Open Sans', a clean sans-serif font for readability.

Colored Callout Example Font

Colored callouts will use 'Lobster', a display font to draw attention to important information.
